*袥懈褌懈褟, 褋芯胁械褉褕邪械屑邪褟 屑懈褉褟薪懈薪芯屑 写芯屑邪 懈 薪邪 泻谢邪写斜懈褖械 薪邪 袪邪写芯薪懈褑褍*

袦芯谢懈虂褌胁邪屑懈 褋胁褟褌褘虂褏 芯褌械虂褑 薪邪虂褕懈褏, 袚芯虂褋锌芯写懈 袠懈褋褍虂褋械 啸褉懈褋褌械虂, 袘芯虂卸械 薪邪褕, 锌芯屑懈虂谢褍泄 薪邪褋. 袗屑懈虂薪褜.

啸褉懈褋褌芯虂褋 胁芯褋泻褉械虂褋械 懈蟹 屑械虂褉褌胁褘褏, / 褋屑械虂褉褌懈褞 褋屑械褉褌褜 锌芯锌褉邪虂胁 / 懈 褋褍虂褖懈屑 胁芯 谐褉芯斜械虂褏 卸懈胁芯虂褌 写邪褉芯胁邪虂胁. *(褌褉懈卸写褘)*

**袠 褌褉芯锌邪褉懈 褋懈褟**, 谐谢邪褋 4: \
小芯 写褍虂褏懈 锌褉邪虂胁械写薪褘褏 褋泻芯薪褔邪虂胁褕懈褏褋褟, / 写褍虂褕懈 褉邪斜 孝胁芯懈虂褏, 小锌邪虂褋械, 褍锌芯泻芯虂泄, / 褋芯褏褉邪薪褟虂褟 懈褏 胁芯 斜谢邪卸械虂薪薪芯泄 卸懈虂蟹薪懈, / 褟虂卸械 褍 孝械斜械虂, 效械谢芯胁械泻芯谢褞虂斜褔械.

袙 锌芯泻芯虂懈褖懈 孝胁芯械虂屑, 袚芯虂褋锌芯写懈, / 懈写械虂卸械 胁褋懈 褋胁褟褌懈虂懈 孝胁芯懈虂 褍锌芯泻芯械胁邪虂褞褌褋褟, / 褍锌芯泻芯虂泄 懈 写褍虂褕懈 褉邪斜 孝胁芯懈虂褏, / 褟虂泻芯 袝写懈虂薪 械褋懈虂 效械谢芯胁械泻芯谢褞虂斜械褑.

**小谢邪胁邪:**\
孝褘 械褋懈虂 袘芯谐 褋芯褕械虂写褘泄 胁芯 邪写, 懈 褍虂蟹褘 芯泻芯胁邪虂薪薪褘褏 褉邪蟹褉械褕懈虂胁褘泄, / 小邪屑 懈 写褍虂褕懈 褉邪斜 孝胁芯懈虂褏 褍锌芯泻芯虂泄.

**袠 薪褘薪械:**\
袝写懈虂薪邪 褔懈虂褋褌邪褟 懈 薪械锌芯褉芯虂褔薪邪褟 袛械虂胁芯, 袘芯虂谐邪 斜械蟹 褋械虂屑械薪械 褉芯虂卸写褕邪褟, / 屑芯谢懈虂 褋锌邪褋褌懈虂褋褟 写褍褕邪虂屑 懈褏.

**袣芯薪写邪泻**, 谐谢邪褋 8:\
小芯 褋胁褟褌褘虂屑懈 褍锌芯泻芯虂泄, / 啸褉懈褋褌械虂, 写褍虂褕懈 褉邪斜 孝胁芯懈虂褏, / 懈写械虂卸械 薪械褋褌褜 斜芯谢械虂蟹薪褜, 薪懈 锌械褔邪虂谢褜, / 薪懈 胁芯蟹写褘褏邪虂薪懈械, / 薪芯 卸懈蟹薪褜 斜械蟹泻芯薪械虂褔薪邪褟.

**袣芯薪写邪泻**, 谐谢邪褋 褌芯泄卸械:\
袗虂褖械 懈 胁芯 谐褉芯斜 褋薪懈蟹褕械虂谢 械褋懈虂, 袘械蟹褋屑械虂褉褌薪械, / 薪芯 邪虂写芯胁褍 褉邪蟹褉褍褕懈虂谢 械褋懈虂 褋懈虂谢褍, / 懈 胁芯褋泻褉械虂褋谢 械褋懈虂, 褟虂泻芯 锌芯斜械写懈褌械谢褜, 啸褉懈褋褌械虂 袘芯虂卸械, / 卸械薪邪虂屑 屑懈褉芯薪芯虂褋懈褑邪屑 胁械褖邪虂胁褘泄: 褉邪虂写褍泄褌械褋褟, / 懈 孝胁芯懈虂屑 邪锌芯虂褋褌芯谢芯屑 屑懈褉 写邪虂褉褍褟泄, / 锌邪虂写褕懈屑 锌芯写邪褟虂泄 胁芯褋泻褉械褋械虂薪懈械.

**袦芯谢懈褌胁邪 芯 褍褋芯锌褕懈褏:**\
袩芯屑褟薪懈虂, 袚芯虂褋锌芯写懈 袘芯虂卸械 薪邪褕, 胁 胁械虂褉械 懈 薪邪写械虂卸写懈 卸懈胁芯褌邪虂 胁械虂褔薪邪谐芯 褍褋芯虂锌褕邪谐芯 褉邪斜邪虂 孝胁芯械谐芯虂 (褍褋芯虂锌褕褍褞 褉邪斜褍虂 孝胁芯褞虂, 褍褋芯虂锌褕懈褏 褉邪斜 孝胁芯懈虂褏) (懈屑械薪邪 懈褏), 懈 褟虂泻芯 斜谢邪谐 懈 褔械谢芯胁械泻芯谢褞虂斜械褑, 芯褌锌褍褖邪虂褟泄 谐褉械褏懈虂, 懈 锌芯褌褉械斜谢褟虂褟泄 薪械锌褉邪虂胁写褘, 芯褋谢邪虂斜懈, 芯褋褌邪虂胁懈, 懈 锌褉芯褋褌懈虂 胁褋褟 胁芯虂谢褜薪邪褟 械谐芯虂 (械褟虂, 懈褏) 褋芯谐褉械褕械虂薪懈褟 懈 薪械胁芯虂谢褜薪邪褟, 胁芯蟹褋褌邪胁谢褟虂褟 械谐芯虂 (褞, 褟) 胁芯 褋胁褟褌芯虂械 胁褌芯褉芯虂械 锌褉懈褕械虂褋褌胁懈械 孝胁芯械虂 胁 锌褉懈褔邪虂褋褌懈械 胁械虂褔薪褘褏 孝胁芯懈虂褏 斜谢邪谐, 懈虂褏卸械 褉邪虂写懈 胁 孝褟 械写懈虂薪邪谐芯 胁械虂褉芯胁邪, 懈虂褋褌懈薪薪邪谐芯 袘芯虂谐邪 懈 褔械谢芯胁械泻芯谢褞虂斜褑邪; 褟虂泻芯 孝褘 械褋懈虂 胁芯褋泻褉械褋械虂薪懈械 懈 卸懈胁芯虂褌, 懈 锌芯泻芯虂泄 褉邪斜褍虂 孝胁芯械屑褍虂 (褉邪斜械虂 孝胁芯械泄, 褉邪斜芯虂屑 孝胁芯懈虂屑) (懈屑械薪邪 懈褏), 啸褉懈褋褌械虂 袘芯虂卸械 薪邪褕, 懈 孝械斜械虂 褋谢邪虂胁褍 胁芯蟹褋褘谢邪虂械屑, 褋芯 斜械蟹薪邪褔邪虂谢褜薪褘屑 孝胁芯懈虂屑 袨褌褑械虂屑, 懈 褋 袩褉械褋胁褟褌褘虂屑 袛褍虂褏芯屑, 薪褘虂薪械 懈 锌褉懈虂褋薪芯 懈 胁芯 胁械虂泻懈 胁械泻芯虂胁, 邪屑懈虂薪褜.

小胁械褌懈虂褋褟, 褋胁械褌懈虂褋褟, / 薪芯虂胁褘泄 袠械褉褍褋邪谢懈虂屑械: / 褋谢邪虂胁邪 斜芯 袚芯褋锌芯虂写薪褟 / 薪邪 褌械斜械虂 胁芯蟹褋懈褟虂. / 袥懈泻褍虂泄 薪褘虂薪械 / 懈 胁械褋械谢懈虂褋褟, 小懈芯虂薪械. / 孝褘 卸械, 效懈虂褋褌邪褟, 泻褉邪褋褍虂泄褋褟, 袘芯谐芯褉芯虂写懈褑械, / 芯 胁芯褋褌邪虂薪懈懈 袪芯卸写械褋褌胁邪虂 孝胁芯械谐芯虂.

小谢邪虂胁邪 袨褌褑褍虂 懈 小褘虂薪褍 懈 小胁褟褌芯虂屑褍 袛褍虂褏褍, 懈 薪褘虂薪械 懈 锌褉懈虂褋薪芯 懈 胁芯 胁械虂泻懈 胁械泻芯虂胁. 袗屑懈虂薪褜.

袚芯虂褋锌芯写懈, 锌芯屑懈虂谢褍泄. *(褌褉懈卸写褘)*

袦芯谢懈虂褌胁邪屑懈 褋胁褟褌褘虂褏 芯褌械虂褑 薪邪虂褕懈褏, 袚芯虂褋锌芯写懈 袠懈褋褍虂褋械 啸褉懈褋褌械虂, 袘芯虂卸械 薪邪褕, 锌芯屑懈虂谢褍泄 薪邪褋, 邪屑懈虂薪褜.

啸褉懈褋褌芯虂褋 胁芯褋泻褉械虂褋械 懈蟹 屑械虂褉褌胁褘褏, / 褋屑械虂褉褌懈褞 褋屑械褉褌褜 锌芯锌褉邪虂胁 / 懈 褋褍虂褖懈屑 胁芯 谐褉芯斜械虂褏 卸懈胁芯虂褌 写邪褉芯胁邪虂胁. *(褌褉懈卸写褘)*

袠 薪邪屑 写邪褉芯胁邪虂 卸懈胁芯虂褌 胁械虂褔薪褘泄, / 锌芯泻谢邪薪褟虂械屑褋褟 袝谐芯虂 褌褉懈写薪械虂胁薪芯屑褍 袙芯褋泻褉械褋械虂薪懈褞.

啸褉懈褋褌芯虂褋 胁芯褋泻褉械虂褋械! *(芯斜褉邪褖械薪懈械 泻 褍褋芯锌褕懈屑)*
